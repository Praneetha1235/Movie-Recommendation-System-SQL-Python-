# Netflix Data Analysis Project

This project analyzes the Netflix Movies and TV Shows dataset to extract insights about content distribution, trends, and genres.

## Tech Stack
Python
SQL
Power BI
Pandas
Matplotlib

## Key Insights
- Movies dominate the Netflix catalog compared to TV shows
- The United States produces the highest amount of content
- Significant growth in content additions after 2016
- Drama and International Movies are the most common genres

## Features
- Data cleaning using Python
- SQL-based analytical queries
- Visualization using Power BI
